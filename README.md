## Installation

To get started, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone <your-repository-url>
   cd <your-project-folder>
   ```
2. **Install dependencies:**
   ```
   npm i
   ```
3. Start the development server:
   ```
   npm run dev
   ```
